

Developement Manual:

This is a Developement Manual for a small TicTacToe program built in Java.

You will need the following three things.

Git
Java
Gradle

Information on how to Install Git https://help.github.com/articles/set-up-git/
Information on how to Install Java https://www.java.com/en/download/help/windows_manual_download.xml
Information on how to Install Gradle https://docs.gradle.org/current/userguide/installation.html

Download Git https://git-scm.com/downloads for appropiate operating system.
Open the git bash and find a spot where you want the project to be.  
use the command:
git clone https://github.com/baldvinj15/TicTacToe.git
That will clone the assignment to you computer.
